# 🤝 行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺，无论年龄、体型、残疾、种族、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性取向如何，参与我们项目和社区的每个人都能享受无骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括：

### ✅ 期望行为

* **使用友好和包容的语言**
* **尊重不同的观点和经验**
* **优雅地接受建设性批评**
* **关注对社区最有利的事情**
* **对其他社区成员表示同理心**
* **耐心帮助新贡献者**
* **感谢他人的贡献和努力**

### ❌ 不当行为

不可接受的行为包括：

* **使用性别化语言或图像**，以及不受欢迎的性关注或性骚扰
* **恶意评论、人身攻击或政治攻击**
* **公开或私下骚扰**
* **未经明确许可发布他人的私人信息**，如物理或电子地址
* **在专业环境中可能被合理认为不当的其他行为**
* **故意散布虚假信息或误导性内容**
* **spam 或无关内容的发布**

## 适用范围

本行为准则适用于所有项目空间，包括但不限于：

* GitHub 仓库（Issues、Pull Requests、Discussions）
* 项目相关的社交媒体账号
* 公开和私下的项目活动
* 项目相关的会议和聚会
* 社区聊天群组和论坛

当个人代表项目或其社区时，本行为准则也适用于公共空间。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发布或在线上或线下活动中担任指定代表。

## 举报和执行

### 🚨 举报不当行为

如果您遇到或目睹了不当行为，请通过以下方式举报：

**首选方式**：
- 📧 邮件：`conduct@godot-mcp.dev`
- 🔒 私密 GitHub Issue：[创建私密 Issue](https://github.com/ee0pdt/godot-mcp/issues/new?labels=conduct&assignees=&template=conduct_report.md)

**举报时请包含**：
- 您的联系方式（可选）
- 涉及人员的描述
- 不当行为的详细描述
- 发生的时间和地点
- 相关证据（截图、链接等）

**我们承诺**：
- 24小时内初步响应
- 72小时内给出处理方案
- 严格保护举报者隐私
- 公正调查所有举报

### ⚖️ 执行措施

项目维护团队有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献，或者暂时或永久禁止任何他们认为有不适当、威胁、冒犯或有害行为的贡献者。

**可能的后果**：
- 📝 私下警告
- 🔒 临时访问限制
- 📢 公开道歉要求
- 🚫 永久禁止参与项目
- 📋 向相关平台报告

### 🛡️ 调查流程

1. **确认收到举报** - 24小时内
2. **初步评估** - 48小时内
3. **收集证据** - 根据需要
4. **做出决定** - 72小时内
5. **执行措施** - 立即执行
6. **跟进检查** - 定期跟进

## 具体场景指导

### 💬 代码审查

**审查者**：
- 提供建设性的反馈
- 解释为什么需要某些更改
- 认可代码中的优点
- 保持耐心和专业

**被审查者**：
- 开放地接受反馈
- 询问不清楚的地方
- 感谢审查者的时间
- 及时回应审查意见

### 🐛 Bug 报告

**报告者**：
- 提供详细的重现步骤
- 包含相关的环境信息
- 保持礼貌和专业
- 配合维护者的调查

**维护者**：
- 及时响应报告
- 请求更多信息时保持耐心
- 解释修复计划和时间线
- 感谢报告者的帮助

### 💡 功能讨论

**提议者**：
- 清晰地描述需求
- 解释使用场景和价值
- 听取其他观点
- 保持开放态度

**参与者**：
- 尊重不同意见
- 提供建设性建议
- 避免人身攻击
- 关注技术而非个人

## 冲突解决

### 🤝 友好解决

如果发生冲突，我们鼓励：

1. **直接沟通** - 首先尝试私下解决
2. **寻求调解** - 请中立的维护者协助
3. **暂时冷静** - 避免在情绪激动时回应
4. **关注问题** - 讨论具体问题而非个人

### 📞 寻求帮助

如果无法友好解决，请：

- 联系项目维护者
- 使用举报渠道
- 寻求社区调解
- 参考 GitHub 的社区准则

## 持续改进

本行为准则是活文档，会根据社区需求不断改进：

* **定期审查** - 每季度审查一次
* **社区反馈** - 欢迎提出改进建议
* **案例学习** - 从实际情况中学习
* **最佳实践** - 参考其他成功开源项目
