# 示例项目

这个目录包含了使用 godot-mcp 进行 AI 自动开发的游戏项目示例，展示了 AI 与 Godot 引擎集成的强大能力。

## 🎮 泡泡游戏演示 (bubble-game-demo)

一个完整的泡泡点击游戏，展示了 godot-mcp 在实际游戏开发中的能力：

### 🌟 展示的功能

#### 1. **智能兼容性修复**
- 自动检测和修复 Godot 3.x 到 4.x 的 API 兼容性问题
- 修复了 Tween 系统、Gradient 系统、粒子系统等多个 API 变化
- 展示了 `godot-compatibility-checker` 技能的实际应用

#### 2. **动态粒子背景系统**
- 15个动态发光粒子，带有缓慢漂浮动画
- 多层次光晕效果和颜色协调
- 展示了 AI 设计复杂视觉效果的能力

#### 3. **精准的得分飘出系统**
- 得分从泡泡位置准确出现并向上飘动
- 专业的 ScorePopup 系统，支持自定义动画
- 展示了 AI 优化用户体验的能力

#### 4. **完整的游戏机制**
- 泡泡生命周期管理
- 点击交互和碰撞检测
- 音效和视觉反馈系统
- 游戏状态管理

#### 5. **智能技能应用**
- **Context7 自动研究**: 自动获取最新的 Godot 文档
- **Godot 兼容性检查**: 实时检测和修复 API 问题
- **MCP 编排**: 协调多个工具完成复杂任务

### 📁 项目结构

```
bubble-game-demo/
├── project.godot              # 项目配置文件
├── GameManager.gd             # 游戏管理器
├── Bubble.gd                  # 泡泡类
├── BackgroundManager.gd       # 动态背景管理器
├── ScorePopup.gd              # 得分飘出效果
├── AudioManager.gd            # 音频管理器
├── EnhancedParticleEffect.gd  # 增强粒子效果
├── ScreenShake.gd             # 屏幕震动效果
├── BubbleGame.tscn            # 主游戏场景
├── Bubble.tscn                # 泡泡预制体
└── README.md                  # 项目说明
```

### 🚀 如何运行

1. 在 Godot 编辑器中打开 `examples/bubble-game-demo/project.godot`
2. 确保已启用 Godot MCP 插件
3. 启动 MCP 服务器: `cd server && npm run dev`
4. 在 Godot 中运行项目 (F5)
